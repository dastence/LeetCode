## 题目思路
暴力思路： 计算每个作为起点计算是否可以到达终点，为了好好记录是否到终点 注意 res 与 index的定义
```java
class Solution {
    public int canCompleteCircuit(int[] gas, int[] cost) {
        int n = gas.length;
        for(int i = 0; i < n; i++){
            int res = gas[i] - cost[i];
            int index = (i + 1)%n;
            while(res > 0 && index !=i){
                res += gas[index] - cost[index];
                index = (index + 1) %n;
            }
            if(res >= 0 && index == i) return i;
        }
        return -1;

    }



}
```
